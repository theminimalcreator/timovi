# What is Timovi?
Timovi gives you a complete digital product team that runs inside your AI
coding agent. Each role has its own skills, responsibilities, and handoffs —
all coordinated through a Plan → Spec → Breakdown → Execute → Review pipeline.

🎯 Product Manager .......... PRDs, priorities, roadmap
🎨 UX Designer .............. Personas, flows, wireframes
🏗️ Software Architect ....... ADRs, feasibility, patterns
👨‍💻 Tech Lead ................ Breakdown, coordination
⚛️ Frontend Engineer ........ Components, pages, state
🖥️ Backend Engineer ......... APIs, services, logic
🗄️ DBA ....................... Schema, queries, indexes
✅ QA Engineer .............. Tests, acceptance criteria

📱 Instagram ................ Social content strategy
# Pipeline: Plan → Spec → Breakdown → Execute → Review
[PLAN] PM + UX Designer + Architect → Plan notes
[SPEC] Product Manager → PRD.md
[BREAKDOWN] Tech Lead + Architect → Issues + feature.json
[EXECUTE] Frontend, Backend, DBA, DevOps → Pull Requests
[REVIEW] QA + Tech Lead + Tech Writer → Approved & merged
