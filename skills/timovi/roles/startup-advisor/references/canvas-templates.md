# Canvas Templates

The Startup Advisor can generate these canvases by asking targeted questions and synthesizing the answers.

## Lean Canvas (Ash Maurya / adapted for this advisor)

```
| PROBLEM              | SOLUTION            | UNIQUE VALUE          | UNFAIR ADVANTAGE     | CUSTOMER SEGMENTS   |
| Top 3 problems       | Top 3 features      | Single, clear,        | Can't be easily      | Target customers    |
|                      |                     | compelling message    | copied or bought     |                     |
|----------------------|---------------------|-----------------------|----------------------|---------------------|
|                      | KEY METRICS         |                       | CHANNELS             |                     |
|                      | Key activities      |                       | Path to customers    |                     |
|----------------------|---------------------|-----------------------|----------------------|---------------------|
| COST STRUCTURE       | REVENUE STREAMS     |                       |                      |                     |
| Customer acquisition | Revenue model       |                       |                      |                     |
| costs, CAC, burn     | Lifetime value      |                       |                      |                     |
```

**Questions to populate**: What problem are you solving? For whom? What's your solution? What makes you unique? How will you reach customers? How will you make money?

## Value Proposition Canvas (Osterwalder / Olsen-adapted)

```
| CUSTOMER PROFILE              | VALUE MAP                      |
|-------------------------------|--------------------------------|
| Jobs to be done:              | Products & services:           |
| - Functional jobs             | - What you offer               |
| - Social jobs                 |                                |
| - Emotional jobs              |                                |
|-------------------------------|--------------------------------|
| Pains:                        | Pain relievers:                |
| - Frustrations with current   | - How you reduce pain          |
| - Risks, obstacles            | - What you eliminate           |
|-------------------------------|--------------------------------|
| Gains:                        | Gain creators:                 |
| - Desired outcomes            | - How you create value         |
| - Surprising delights         | - What exceeds expectations    |
```

**Questions to populate**: What jobs is the customer trying to do? What pains do they feel? What gains do they want? How does your product relieve pains and create gains?

## Business Model Canvas (Osterwalder)

```
| KEY PARTNERS         | KEY ACTIVITIES       | VALUE PROPOSITION     | CUSTOMER RELATIONSHIPS | CUSTOMER SEGMENTS   |
|----------------------|---------------------|-----------------------|------------------------|---------------------|
| KEY RESOURCES        |                     |                       | CHANNELS               |                     |
|----------------------|---------------------|-----------------------|------------------------|---------------------|
| COST STRUCTURE       | REVENUE STREAMS     |                       |                        |                     |
```

**Questions to populate**: Same as Lean Canvas + Who are your key partners? What activities are critical? What resources do you need?

## PMF Pyramid (Olsen)

```
    5. UX (User Experience)
   4. Feature Set (MVP scope)
  3. Value Proposition (promise)
 2. Underserved Needs (gap)
1. Target Customer (who)
```

**Questions to populate**: Who is the target customer? What needs are underserved? What's the value proposition? What features deliver it? What's the UX?

## Growth Funnel Canvas

```
Acquisition → Activation → Retention → Revenue → Referral
     |            |            |           |          |
  [metric]    [metric]     [metric]    [metric]  [metric]
  [channel]   [barrier]    [barrier]   [model]   [loop]
```

**Questions to populate**: How do users find you? What's activation? What's retention? How do you monetize? Do users refer others?
